About the Project

The Webserv project involves creating a high-performance HTTP server in C++98 that implements core HTTP protocol functionality. This custom server handles multiple ports, supports various HTTP methods (GET, POST, DELETE), manages file uploads, executes CGI scripts, and maintains non-blocking I/O operations for optimal performance.

Core Features

🚀

HTTP/1.1 Compliant

Full implementation of HTTP 1.1 protocol with support for GET, POST, and DELETE methods. Compatible with all modern web browsers.

⚡

Non-Blocking Architecture

Uses poll() or equivalent system calls to handle all I/O operations without blocking, ensuring maximum efficiency and scalability.

🔧

Flexible Configuration

Customizable through configuration files to set ports, server names, error pages, client limits, and route-specific rules.

📁

Advanced File Handling

Serves static content, handles file uploads, manages directory listings, and supports default index files.

🛡️

Robust Error Handling

Comprehensive error pages and accurate HTTP status codes. Designed to remain stable under heavy load.

🔗

CGI Integration

Executes CGI scripts (PHP, Python, etc.) with proper environment setup and directory context.
